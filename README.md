# React Practice 3

React Practice 3 — учебный проект, созданный на основе React. Разработан в рамках изучения фронтенд-разработки для практики построения одностраничных приложений (SPA), работы с компонентами, состоянием и маршрутизацией.



## Описание проекта

React Practice 3 представляет собой небольшое SPA-приложение, созданное для демонстрации базовых принципов работы с библиотекой React.
Проект разработан в учебных целях и подходит начинающим фронтенд-разработчикам, осваивающим React и основы клиентской логики.

Основные функции:

Создание и использование функциональных компонентов

Управление состоянием с помощью хуков (useState, useEffect)

Обработка пользовательского ввода и событий

Маршрутизация с помощью react-router-dom

Базовая структура React-проекта


## Технологии
В проекте используются современные инструменты для разработки и сборки React-приложений.

Основной стек:

React 18+ — библиотека для создания пользовательских интерфейсов

React Router DOM — для маршрутизации между страницами

JavaScript (ES6+) — язык разработки

npm — менеджер пакетов

Vite / CRA (Create React App) — для запуска и сборки проекта



##  Установка и запуск

Технические требования
| Компонент               | Версия / Требования                     |
| ----------------------- | --------------------------------------- |
| Язык разработки         | JavaScript (ES6+)                       |
| Фреймворк               | React 18+                               |
| Менеджер пакетов        | npm 9+ / yarn                           |
| Среда выполнения        | Node.js v18 или выше                    |
| Операционные системы    | Windows / macOS / Linux                 |
| Поддерживаемые браузеры | Актуальные версии Chrome, Firefox, Edge |


1. **Склонировать репозиторий:**
   git clone  https://github.com/tatianapudow/react-practice3.git
   cd my-vite-app
2. **Установить зависимости**   
    npm install
3. **Запустить приложение в режиме разработки**   
    npm start
После запуска приложение будет доступно по адресу:
 http://localhost:3000
